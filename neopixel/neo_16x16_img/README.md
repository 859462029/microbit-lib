# micro:bit neopixel 16x16 display drive

![](microbit.gif)


## API

* **clear()**  
clear all LED  

* **show(self, dat, offset=0)**  
show color emijo.  
dat:    color emijo data  
offset: display offset  


![](neo16x16.jpg)  

**Please note neopixel display must using external power.**
